# Radar Local v0.1.0

Primeira versao publica do Radar Local como projeto open source.

## O que ja esta disponivel

- busca de negocios locais por regiao, categoria e raio
- score comercial com sinais de maturidade digital
- oferta sugerida, urgencia e mensagem inicial por lead
- demo publica por nicho para apoio comercial
- pipeline simples para acompanhar abordagem e proximos passos
- persistencia local e suporte opcional a Supabase

## Como testar localmente

```bash
npm install
npm run setup
npm run dev
```

Depois, abra `http://localhost:3000`.

## Observacoes desta release

- o projeto funciona em modo local mesmo sem todas as chaves externas
- alguns fluxos de compartilhamento ainda dependem de configuracao complementar
- a direcao atual do produto e prospeccao orientada a fechamento para negocios locais

## Proximos focos

- fila operacional do dia
- score de vendabilidade mais forte
- mais confiabilidade nas demos por nicho
- metricas de ROI e acompanhamento comercial
